These are 5 HTML Tags I have learned and thier description

Placeholder
The placeholder attribute is used to display placeholder text inside an input field.
Value
The value attribute is used to pre-fill the value of an input field. This is useful when we are getting the previously saved
information by the used from the database and then displaying it to the user for editing in a form.
Check Boxes
A small multi-selection grouping. More than one selection can apply.
Merging cells
merge the adjacent cells in columns or rows by controlling the number of columns and rows each table data cell in the table
takes up using the colspan and rowspan attributes.
Table data Cell
The tag represents the data cell in the table. the table data cells are adding inside the table row tags for each piece of data.
